# Pizza Menu 🍕

A beautiful and responsive pizza restaurant menu built with React. 

## Features
- Fully responsive design (mobile, tablet, desktop)
- Displays pizza list with name, ingredients, price and photo
- Shows "Sold Out" label for unavailable pizzas
- Clean and modern UI with CSS modules
- Data stored in a static array (perfect for practicing .map(), conditional rendering, and props)

## Technologies Used
- React 18 (Functional Components + Hooks)
- useState for managing sold-out state
- CSS Modules for styling
- Vite as build tool

## What I Learned
- Component composition and reusability
- Conditional rendering and logical operators
- Passing props and rendering lists with `.map()`
- Proper key usage in lists
- Responsive design without any framework
